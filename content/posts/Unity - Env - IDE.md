## Configuration - VS Code

### .NET SDK 安裝

1. 前往 [.NET 官方下載頁面](https://dotnet.microsoft.com/en-us/download) 下載對應版本
2. 選擇對應平台的 SDK：
    - **Mac M 系列 (M1/M2)** ➔ 安裝 **Arm64** 版本
    - **其他 Mac 型號** ➔ 安裝 **x64** 版本
    - **Windows** ➔ 安裝 **x64** 版本
3. 安裝完成後 **重新啟動電腦**，以讓環境變數生效
