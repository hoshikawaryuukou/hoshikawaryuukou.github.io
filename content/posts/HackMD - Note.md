## Quick Chat
HackMD 的核心思想是
> 輕鬆使用 Markdown 協作文件的工具

符合的我的使用需求
- 免費方案適合輕度用戶
- 跨平台兼容性（桌面/移動設備）
- 支援 Markdown 格式
- 文件管理（標籤/搜索功能）
- 分享控制（私人/公開）
- 支援遠端同步

付費方案差異
- 多人協作人數
- 文件大小限制
- 版本控制
- API 接入
