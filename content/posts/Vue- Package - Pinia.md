## Why Use It?
- 官方支持 : Pinia 是 Vue 官方推薦的狀態管理解決方案，與 Vue 3 完全兼容，並且擁有更現代的 API 設計。
- TypeScript 原生支持 : 自動推斷 store 中的狀態、getters 和 actions 的類型，不需要手動編寫冗長的類型聲明。
- 支持 DevTools : 支持 Vue DevTools 的狀態追蹤功能。
- 支持熱重載 : 即時修改 store，而不會丟失應用程序當前的狀態。
