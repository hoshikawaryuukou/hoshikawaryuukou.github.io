## Quick Chat
nvm-windows 是一款 Windows 作業系統下的 Node.js 版本管理器，可讓使用者在同一台電腦上安裝、切換和管理多個不同的 Node.js 版本。

## Guide
- [coreybutler/nvm-windows](https://github.com/coreybutler/nvm-windows)
- [nvm-windows 管理 Windows Node.js 版本](https://medium.com/@ray102467/nvm-windows-%E7%AE%A1%E7%90%86-windows-node-js-%E7%89%88%E6%9C%AC-68d789cf84d7)

## Commands
- nvm -v : 查看 nvm 版本
- nvm install lts : 安裝最新的 Node.js LTS 版本
- nvm install {version} : 安裝版本
- nvm use {version} : 切換版本
