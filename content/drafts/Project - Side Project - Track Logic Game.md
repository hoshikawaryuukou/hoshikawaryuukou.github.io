## Rule

1. **玩家與棋盤：**
   - 遊戲由兩名玩家進行，分別為 `'X'` 和 `'O'`。
   - 棋盤為 4x4 的格子，初始時所有格子均為空白。
   - 玩家輪流在空格位置上放置自己的棋子（`'X'` 或 `'O'`）。

2. **遊戲流程：**
   - **放置棋子**：每位玩家在自己的回合選擇一個空格，放下自己的棋子。
   - **旋轉棋盤**：每次放置棋子後，棋盤上的所有棋子**逆時針旋轉一格**。
   - **勝利判定**：若某位玩家的四顆棋子連成一線（橫向、縱向或對角線），則該玩家獲勝。若無勝負且仍有空位，則換另一位玩家繼續。
