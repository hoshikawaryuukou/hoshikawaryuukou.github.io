Electron - Note

2024-10-11
Electron

Quick Chat

Electron 是一個使用 JavaScript、HTML 和 CSS + Native Api 做相容多個系統（Windows、Linux、Mac）的桌面應用程式構建框架 —— 不需要本地開發經驗。

Guide

  • Build cross-platform desktop apps with JavaScript, HTML, and CSS | Electron
  • electron-builder
  • Electron学习指引
  • awesome-electron
  • awesome-electron-alternatives

Suggestion

🤓 強烈建議第一次接觸 Electron 的開發者實際走一次官方 quick-start，會對 Electron 的本質有更純粹的認識。

  • electron/electron-quick-start

再逐步的嘗試以下機制 :

  • 用 BrowserWindow 再額外創建一個窗口。
  • 用 ipcMain/ipcRenderer 來達到進程間交互。

Notice

  • ⚠️ Electron 中的 ES 模塊 (ESM) 並未 100% 支援
