Unity - Package Manager - Modify A Package

2024-01-12
Unity , Package Manager

情境

有需要客製化 Unity 官方 / git 直接導入 的 Package 時

操作

  1. 至 ProjrctRoot\Library\PackageCache 找到目標的 Package
  2. 剪下貼上至 ProjrctRoot\Packages 即可
