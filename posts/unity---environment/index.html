Unity - Environment

2024-01-16
Unity , Mac , VScode

⚠️ 以下使用 unity 6

OS - Mac

  • 【游戏开发环境】Unity使用Mac电脑开发，开发环境的搭建（Mac mini M1 | VSCode | Git | 好用工具）

VSCode

  • 安裝 .NET Core
  • 重新開機使環境變數被應用
  • 移除 Visual Studio Code Editor 已經停止維護
  • 安裝 Visual Studio Editor
  • Edit > Preferences > External Tools > External Script Editor
  • Extension - C# Dev Kit

Preferences

  • Edit > Preferences > General > Script Changes While Playing (Stop Playing and Recompile)
