Node.js - NPM - Package Creating

2024-07-26
Node.js

Configuration - tsconfig.json

  • @tsconfig/node20 : Node 20 推薦的 tsconfig.json
  • "declaration" : 是否為這個庫提供類型支持。
  • "outDir" : 聲明編譯後的 JavaScript 文件和類型聲明文件的輸出目錄。
  • "include" : 聲明參與編譯過程中的文件和目錄。
{
  "extends": "@tsconfig/node20/tsconfig.json",
  "compilerOptions": {
    "module": "commonjs",
    "target": "es2015",
    "declaration": true,
    "outDir": "./dist"
  },
  "include": [
    "src/**/*"
  ]
}

Configuration - package.json - Entry Points

  • Multiple exports with types in a Typescript Package
  • package.json 中的 exports、main、module 字段

Single export

{
  "main": "dist/index.js",
  "types": "dist/index.d.ts",
}

Multiple exports

{
  "exports": {
    ".": "./dist/index.js",
    "./engine": "./dist/engine.js"
  },
  "typesVersions": {
    "*": {
      "*": ["./dist/*"]
    }
  }
}

Commands

Version

可以使用以下命令修改 package.json 中的版本號

  • npm version patch : 補丁/錯誤修復時
  • npm version minor : 新功能
  • npm version major : 重大更改

Build

  • npm run build : 編譯 ts 至 tsconfig.json 指定的路徑
