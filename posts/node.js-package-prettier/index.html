Node.js - Package - Prettier

2024-07-02
Node.js

Quick Chat

Prettier 是一個代碼格式化工具，用於保持代碼風格一致。

Guide

  • Prettier · Opinionated Code Formatter
  • 如何配置 prettier

Install

pnpm add -D prettier

Configure

  • .prettierrc : Configuration File · Prettier
  • .prettierignore : Ignoring Code · Prettier

Plugins

  • prettier-plugin-organize-imports : 排序/合併/移除未使用的 import 聲明
{
  "plugins": ["prettier-plugin-organize-imports"]
}

Issue

當跨平台協同的時候會有結尾符的問題

  • [eslint] Delete CR eslint(prettier/prettier) issue
  • 令人困擾的git autocrlf – Opass Life

於 .prettierrc 新增參數

{
  "endOfLine": "auto"
}

如果已經啟用 auto format，但並未生效

可能是有多個 formatter 同時作用，需指定預設 formatter (vscode 右下角應該會提示要選擇一種為預設)
