Node.js - Package - Prettier

2024-07-02
Node.js

Quick Chat

Prettier 是一個代碼格式化工具，用於保持代碼風格一致。

Guide

  • Prettier · Opinionated Code Formatter

Install

npm install -D prettier eslint-plugin-prettier eslint-config-prettier
  • eslint-plugin-prettier : ESLint 插件，用於將 Prettier 內嵌到 ESLint 中，使 ESLint 可以報告 Prettier 的格式錯誤。
  • eslint-config-prettier : ESLint 配置，用於關閉所有 ESLint 中的格式相關規則，以避免與 Prettier 產生衝突。
